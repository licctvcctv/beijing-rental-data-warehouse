# 原始采集字段映射

## 通用要求

- 编码统一为 UTF-8
- 空值统一写 `\\N`
- 所有导出 CSV 都追加 `source_url`、`source_site`、`crawl_time`
- 采集阶段只做轻量清洗，不做业务标准化

## scenic_raw.csv

当前优先来源已扩展为北京旅游网资源库 `休闲娱乐(type=8)` 列表分页 + 详情页；当主源异常偏少时，再回退补充北京市文旅局官方景点链接清单。

| 字段 | 说明 | 当前来源 |
|---|---|---|
| name | 景点名称 | 北京旅游网资源库列表标题 / 官方清单名称列 / 详情页标题 |
| level | 景点等级 | 详情页景区等级，或官方清单等级列 |
| region | 所在区域 | 详情页地址提取，或官方清单区域列 |
| address | 地址 | 详情页“地址” |
| price | 门票价格 | 详情页“门票价格” |
| open_time | 开放时间 | 详情页“开放时间” |
| visit_duration | 推荐游览时长 | 详情页“推荐游览时长” |
| best_visit_time | 最佳游览时间 | 详情页“最佳游览时间” |
| source_url | 详情页链接 | 详情页 URL |
| source_site | 来源站点 | 北京旅游网 / 北京市文旅局 |
| crawl_time | 抓取时间 | 程序运行时写入 |

## show_raw.csv

当前来源为两段组合：
- 大河票务北京页 + 详情页，保留售票状态和关注度
- 东方演出网北京城市页列表卡片，补足更大体量的北京演出数据

为控制单次抓取时长，东方演出网默认只抓取北京城市页前 20 页，并与大河票务结果按名称/时间/场馆去重后导出。

| 字段 | 说明 | 当前来源 |
|---|---|---|
| name | 演出名称 | 大河票务详情页标题 / 东方演出网列表标题 |
| show_time | 演出时间 | 大河票务详情页时间块 / 东方演出网列表“演出时间” |
| venue | 场馆 | 大河票务详情页场馆块 / 东方演出网列表“地点” |
| region | 所在区域 | 当前固定写北京 |
| price_range | 票价区间 | 大河票务详情页价格列表 / 东方演出网列表“门票价格” |
| status | 售票状态 | 大河票务详情页状态标签；东方演出网当前写 `\\N` |
| attention | 关注度 | 大河票务详情页关注/热度值；东方演出网当前写 `\\N` |
| source_url | 详情页链接 | 来源详情页 URL |
| source_site | 来源站点 | 大河票务 / 东方演出网 |
| crawl_time | 抓取时间 | 程序运行时写入 |

## ktv_raw.csv

当前来源为城市惠北京 `KTV` + `商务KTV` 两条榜单页与详情页组合采集，并按地址优先去重，避免同一会所被不同文章重复写入。

| 字段 | 说明 | 当前来源 |
|---|---|---|
| name | KTV 名称 | 详情页标题 |
| region | 所在区域 | 优先由地址提取 |
| address | 地址 | 详情页“地址” |
| avg_cost | 人均/参考价格 | 详情页“参考价格”或“消费价格”提取数字 |
| service_score | 服务/口碑分 | 详情页“口碑评分” |
| env_score | 环境分 | 详情页显式“环境评分”，无则 `\\N` |
| overall_score | 综合评分 | 详情页“综合评分” |
| popularity | 人气指数 | 详情页“人气指数” |
| business_hours | 营业时间 | 详情页“营业时间” |
| source_url | 详情页链接 | 城市惠详情页 URL |
| source_site | 来源站点 | 城市惠 |
| crawl_time | 抓取时间 | 程序运行时写入 |

## movie_raw.csv

当前来源已扩展为豆瓣电影 `Top250` 列表分页批量采集，直接在列表页提取名称、评分、导演/主演摘要、国家地区、类型和简介，避免详情页反爬导致的稳定性问题。

| 字段 | 说明 | 当前来源 |
|---|---|---|
| name | 电影名称 | 豆瓣 Top250 列表项标题 |
| score | 豆瓣评分 | 列表项评分 |
| category | 电影类型 | 列表项年份/地区/类型摘要中的类型段 |
| country_region | 国家/地区 | 列表项年份/地区/类型摘要中的地区段 |
| director | 导演 | 列表项演职员摘要中的导演字段 |
| actors | 主演 | 列表项演职员摘要中的主演字段 |
| intro | 简介短评 | 列表项短评 |
| source_url | 详情页链接 | 豆瓣电影详情页 URL |
| source_site | 来源站点 | 豆瓣电影 |
| crawl_time | 抓取时间 | 程序运行时写入 |

## sport_raw.csv

当前优先来源为城市惠北京运动类榜单（健身房/滑雪场/瑜伽馆）+ 详情页组合采集；当主源可发现链接过少时，再回退补充北京旅游网资源库中的体育相关条目。

| 字段 | 说明 | 当前来源 |
|---|---|---|
| name | 场馆/条目名称 | 详情页标题 |
| venue_type | 场馆类型 | 城市惠榜单类别或详情页显式标签 |
| region | 所在区域 | 详情页地址提取 |
| address | 地址 | 详情页“地址” |
| score | 评分 | 城市惠“综合评分”或北京旅游网默认 `\\N` |
| comment_count | 评论数 | 当前未稳定提供，先写 `\\N` |
| avg_cost | 人均费用 | 城市惠“参考价格”提取数字，缺失则 `\\N` |
| open_time | 开放时间 | 详情页“营业时间”或“开放时间” |
| source_url | 详情页链接 | 详情页 URL |
| source_site | 来源站点 | 城市惠 / 北京旅游网 |
| crawl_time | 抓取时间 | 程序运行时写入 |
